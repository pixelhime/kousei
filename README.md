# 校正 — ホラーノベル

――消セ。

あなたは、鵜山書房 校閲部の校閲者。回収された曰くつきの原稿を、墨で塗り消して世に送り出すのが仕事だ。
選択肢は、ない。**「消す」ことだけ**が、あなたの意思になる。

墨は、戻らない。そして、あなたが消したことばは——どこへ行くのだろう。

「消す」という行為そのものを仕掛けにした、連作ホラーノベルゲーム。**原稿三本・全九つの結末**を校了せよ。

## 遊ぶ

公開URL:
`https://pixelhime.github.io/kousei/`

または `index.html` をブラウザで直接開いても遊べます（単一HTMLファイル・ビルド不要）。

## 遊びかた

- 縦書きの原稿を読み、消すべきことばを **クリック** して墨を入れる。
- 各章には校閲指示がある。**対象の語をすべて消す**と提出できる。
- 墨は戻せない。何を消し、何を残すかで、結末が変わる。

## 特徴

- 受け取り箱に届く **連作三部作** — 一本を校了すると、次の原稿が届く
  1. 『雨の家』 — 雨の絶えない家の、妻の日記
  2. 『供述』 — 妻殺しで起訴された夫の、獄中手記
  3. 『無題』 — 差出人も題もない、あなたの消したことばでできた原稿
- 消しかたで分岐する **全九つの結末**（到達状況を受け取り箱に記録）
- あなたが墨に沈めたことばが、章をまたいで **持ち越される** 仕掛け
- 縦書き原稿用紙の校正UI・墨入れ演出
- WebAudio による雨音（オン/オフ切替）
- オートセーブ（localStorage）／スマホ対応
- **ヒント機能（ver 1.1）** — 初回クリア後、受け取り箱の未取得「？」から段階式の校閲メモ（ヒント）を確認できる。結末をすべて消して最初から遊べるリセットも搭載

## 技術

依存ライブラリなし。`index.html` 一枚で完結。
